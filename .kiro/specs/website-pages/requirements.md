## Glossary

- **Website**: The New Way Solutions corporate website built with Next.js
- **Page**: A standalone route accessible via URL (e.g., /services, /about)
- **Component**: Reusable React component used across pages
- **Color Scheme**: The existing teal/cyan gradient color palette used throughout the site
- **Navigation**: The header navigation that links to all pages
- **Responsive Design**: Layout that adapts to mobile, tablet, and desktop screen sizes
